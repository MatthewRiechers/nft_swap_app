- State Management (Bloc & Cubit)
- Networking (okHttp)
- Api Data (MockApi)
- Architecture: Model -> Repository -> State Management (Bloc) -> Presentation  
- Proper error and exception handling
- Navigation (GoRouter)
- Splash Screen With Some Cool Animation
